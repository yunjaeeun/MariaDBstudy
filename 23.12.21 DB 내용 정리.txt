[데이터베이스란]

데이터:자료, 정돈되지 않은것
인포메이션:데이터로 부터 정돈되어진 것 
데이터는 정량만 따진다면 인포메이션은 정성도 따짐

데이터베이스:데이터가 인포메이션화 될 수 있도록 잘 정돈한 저장소
기본적으로 영구적 저장 목적임. 데이터 베이스는 보조 기억장치에 저장됨(비 휘발성메모리 ex) hdd ssd)

DBMS
데이터 베이스가 창고라고 한다면 DBMS가 창고를 관리하는 시스템이라고 생각 하면 됨
MariaDB는 DBMS의 종류 중 하나
명령은 쿼리를 이용해서 함 DBMS에게 요청을 해 자료를 가져옴

[기능]
데이터를 조회해 추출 가능(데이터 추출)
데이터을 만들거나 수정, 삭제 가능(데이터 조작)
데이터를 놔둘 공간과 정리 방식을 정리(데이터 정의)

[사용이점]
무결성:확신을 가지고 어떤 데이터를 가지고 있는지(뽑을 때 마다 원하는 데이터가 나오게)
표준화:DB마다 관리법이 다르면 사용하기 어려우므로 방법을 표준화 시킴.

관계형 데이터베이스(RDBMS)
표들이 그냥 존재하는게 아닌 관계를 가지고 존재한다.

1. 서로 다른 테이블들을 한번에 조회하기 위함이 가장 큼
